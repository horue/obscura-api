Obscura API

Micro-API para integrar a newsletter Obscura em qualquer frontend

GET /api/health

Verifica se a API está funcionando.

// Resposta
{ "status": "ok", "timestamp": "2026-05-28T00:00:00.000Z" }
POST /api/subscribe

Inscreve um email na newsletter Obscura (Substack).

// Request (JSON)
{ "email": "usuario@email.com" }

// Resposta (sucesso)
{ "ok": true, "status": 200 }

// Resposta (erro)
{ "error": "Email inválido" }

Body: {"email": "usuario@email.com"}

GET /api/feed

Retorna os 3 posts mais recentes da Obscura (substituto do rss2json, sem CORS).

// Resposta
{
  "ok": true,
  "items": [
    {
      "link": "https://www.aobscura.com.br/p/titulo",
      "title": "Título do Post",
      "desc": "Descrição...",
      "img": "https://...",
      "tag": "Categoria",
      "meta": "Mai 2026 · Leia na Obscura →"
    }
  ]
}

Respeita o header Cache-Control: public, max-age=1800 (30 min).

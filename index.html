Obscura API

Micro-API para integrar a newsletter Obscura em qualquer frontend

GET /api/health

Verifica se a API está funcionando.

{ "status": "ok", "timestamp": "2026-05-28T00:00:00.000Z" }
POST /api/subscribe

Inscreve um email na newsletter Obscura (Substack).

// Request
{ "email": "usuario@email.com" }

// Resposta
{ "ok": true }

Body: {"email": "usuario@email.com"}

GET /api/feed

Retorna os 3 posts mais recentes da Obscura em JSON (sem CORS).

// Resposta
{
  "ok": true,
  "items": [
    {
      "link": "https://...",
      "title": "Título do Post",
      "desc": "Descrição...",
      "img": "https://...",
      "tag": "Categoria",
      "meta": "Mai 2026 · Leia na Obscura →"
    }
  ]
}

Cache de 30 min. Use a URL base da API no lugar de https://obscura-api-amber.vercel.app.

Exemplo 1 — Lista simples (copiar e colar):

<div id="obscura-feed">Carregando...</div>
<script>
fetch('https://obscura-api-amber.vercel.app/api/feed')
  .then(function(r){ return r.json(); })
  .then(function(d){
    if (!d.ok) return;
    var html = '';
    d.items.forEach(function(p){
      html += '&lt;div style="margin-bottom:16px"&gt;' +
        (p.img ? '&lt;img src="' + p.img + '" style="width:100%;max-width:400px"&gt;' : '') +
        '&lt;h3&gt;&lt;a href="' + p.link + '"&gt;' + p.title + '&lt;/a&gt;&lt;/h3&gt;' +
        '&lt;p&gt;' + p.desc + '&lt;/p&gt;' +
        '&lt;small&gt;' + p.meta + '&lt;/small&gt;' +
        '&lt;/div&gt;';
    });
    document.getElementById('obscura-feed').innerHTML = html;
  });
</script>

GET /api/widget

Retorna um HTML pronto para incorporar os 3 posts em qualquer site. Basta colocar num <iframe>.

// Parâmetros (todos opcionais)
bg      = 0d1318   cor de fundo
text    = e0e0e0   cor do texto
accent  = f44336   cor de destaque
card    = 1c2126   cor do card
border  = 252b2f   cor da borda
muted   = 9ea1a3   cor secundária
title   =           título opcional
font    =           fonte (ex: Georgia,serif)
layout  = vertical  (padrão) ou horizontal

// Vertical (padrão) — cards empilhados
<iframe src="https://obscura-api-amber.vercel.app/api/widget?bg=0d1318&text=ffffff&accent=f44336&title=Obscura"
  width="100%" height="520" style="border:0;border-radius:12px"></iframe>

// Horizontal — 3 cards lado a lado, responsivo
<iframe src="https://obscura-api-amber.vercel.app/api/widget?layout=horizontal&bg=0d1318&text=ffffff&accent=f44336"
  width="100%" height="360" style="border:0;border-radius:12px"></iframe>

O widget já vem com CSS inline — funciona em qualquer lugar, inclusive em sites que bloqueiam JavaScript.
